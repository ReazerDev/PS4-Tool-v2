# How To Use

### Payload Sender

-Go into Settings and type in your PS4's IP Address

-Open an Exploit page on your PS4

-Send the selected Payload



### Fan Control

-Go into Settings and type in your PS4's IP Address

-Open an Exploit page on your PS4

-Choose the temperature when your Fan should start spinning, by moving the slider

-Click on Send
